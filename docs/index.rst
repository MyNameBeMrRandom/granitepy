
Welcome to granitepy's documentation
=====================================
granitepy is a python wrapper for the audio provider called `andesite <https://github.com/natanbc/andesite-node/>`_
for use with `discord.py <https://github.com/Rapptz/discord.py/>`_. For help and support using granitepy, you can
join the `support server <https://discord.gg/xP8xsHr>`_ on `discord <https://discordapp.com/>`_.



Installation:
-------------

From PyPi (Stable version):

.. code:: sh

    pip install -U granitepy


From Github (Development version):

.. code:: sh

    pip install -U git+https://github.com/iDevision/granitepy
